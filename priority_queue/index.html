Priority Queue Load Balancer Visualization

Priority Queue Algorithm:
This load balancer assigns each new request to the worker with the lowest current load (total processing time). The workers are sorted by their current load in a priority queue, and the one with the minimum load receives the next request. This minimizes wait times and ensures even distribution of work.
